Cave Family Accounts

User Instructions — how to enter, save, and manage monthly account data

Overview

The app tracks the Cave family's bank account balances and credit card debits each month. All data is saved locally in your browser — nothing is sent to the internet. Each month is stored as a separate record and you can navigate between them using the arrows at the top of the page.

Toolbar Buttons

+ New Month

Creates a record for the next calendar month. The current month must be saved first. The new month starts with the same account values copied across, and the carry-over figure is set to this month's net position so the savings calculation is correct.

Save

Locks the current month and marks it as saved. The status badge in the top-right of the toolbar changes from Editing (orange) to Saved (green). You must save a month before creating a new one.

Edit

Unlocks a previously saved month so you can make corrections. The Edit button is only visible when a month is in Saved state.

Export JSON

Downloads a complete backup of all months as a .json file. Keep a copy somewhere safe (e.g. a USB drive or cloud folder) so your data is not lost if you clear your browser or change device.

Import

Lets you load data into the app from two sources:

  • JSON backup — select a previously exported .json file. All months in the backup will be merged in. Existing months are not overwritten.
  • Google Sheets CSV — download your Google Sheet as a .csv file (File → Download → CSV) and select it. You will be asked which month (YYYY-MM) the data belongs to.

Delete Month

Permanently removes the currently displayed month from the app. A confirmation prompt appears before anything is deleted. Make sure you have a JSON backup if you may need the data later.

Month Navigation

Use the ← → arrows in the header to move between months. The arrows are disabled when you are already at the oldest or most recent month.

Tips

Regular backups: Use Export JSON at the end of each month and save the file somewhere outside the browser (USB drive, email to yourself, cloud storage). Browser storage can be cleared accidentally.
Kath's share field: Only update this when the split between Kath and Amber actually changes — you don't need to re-enter it every month as the New Month button copies it forward automatically.
Importing old Google Sheets data: The app recognises both current and older account names from pre-2024 sheets, so historical CSVs should import cleanly. Closed accounts (Halifax, NS&I, safe accounts) are silently ignored.
